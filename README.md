# AgriLink AI 🌾

AgriLink AI is a prototype agriculture platform designed to help farmers make better selling decisions, reduce crop waste, and improve transparency in agricultural trade.

The platform combines practical AI-assisted features such as crop disease guidance, price estimation, and an agriculture chatbot with a marketplace flow for selling crops and agricultural waste. It also includes a transaction transparency layer to demonstrate how trust and traceability can be improved in farmer-to-buyer interactions.

## Why this project exists

Many farmers still face common challenges such as:

* unclear or unfair crop pricing,
* crop loss due to delayed disease identification,
* limited direct access to buyers,
* and crop waste being burned instead of reused.

This project was built to explore a more connected and transparent system where farmers can not only sell produce more effectively, but also generate value from agricultural waste that is often discarded.

## What the platform includes

* **Crop price insight module** to help farmers estimate reasonable selling value
* **Disease guidance flow** for identifying possible crop health issues
* **Agriculture chatbot** for basic assistance and recommendations
* **Crop marketplace** for farmer-to-buyer listing flow
* **Crop waste marketplace** to encourage reuse of biomass and reduce pollution
* **Transaction transparency dashboard** inspired by ledger-based record keeping
* **Role-based dashboards** for farmers, buyers, and admin users

## How blockchain is used in this project

This project does **not claim to be a fully deployed blockchain product**.

Instead, it uses a **blockchain-inspired transaction transparency model** to represent how agricultural transactions can be recorded in a more traceable and trust-oriented way. The current prototype focuses on the user flow and system design, with room for future integration of real smart contracts or decentralized infrastructure.

## How AI is used in this project

The AI layer is used to simulate or support:

* crop disease-related suggestions,
* crop pricing guidance,
* and conversational support through the chatbot.

The current version is designed as a **hackathon prototype**, so the focus is on demonstrating practical use cases and product flow rather than building production-grade agricultural ML systems.

## Project Goal

The goal of AgriLink AI is not to overcomplicate farming with technology, but to show how simple digital tools can improve:

* decision-making,
* price transparency,
* waste utilization,
* and trust in agricultural trade.

## Tech Stack

* React / React Native / Expo
* TypeScript
* Mock datasets / lightweight app logic
* Gemini-assisted chatbot or AI simulation
* UI-driven transparency workflow

## Note

This is a prototype built for experimentation and demonstration purposes, with emphasis on usability, real-world relevance, and social impact.
